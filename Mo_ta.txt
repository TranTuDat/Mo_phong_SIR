🧩 1. CẤU TRÚC TỔNG THỂ GIAO DIỆN

Trang web được thiết kế theo mô hình Dashboard phân tích dữ liệu đa lớp, bố cục dạng:

| Sidebar |        Main Content (Dashboard)        |
|         |  Topbar                              |
|         |  Metrics Cards                       |
|         |  Network Graph + Table               |
|         |  Detail Panels                       |
🔹 Đặc điểm chung:
Layout dạng 2 cột cố định
Sidebar cố định bên trái
Nội dung bên phải dạng grid linh hoạt
Thiết kế theo hướng:
Data-centric
Visualization-first
Decision-support system
🧭 2. SIDEBAR (THANH ĐIỀU HƯỚNG TRÁI)
📌 2.1. Tổng quan thiết kế
Chiếm khoảng 18–22% chiều rộng màn hình
Nền: xanh navy đậm (#0B1F3A gần đúng)
Font chữ: trắng / xám nhạt
Phân cấp rõ ràng bằng:
Icon
Khoảng cách dòng
Highlight trạng thái
📌 2.2. Header sidebar
Icon logo (hình khiên + biểu tượng phân tích)
Tên hệ thống:
InfoOps Analyzer (font đậm, lớn)
Subtitle:
“Công cụ phân tích mạng xã hội”
Căn trái hoàn toàn
📌 2.3. Menu điều hướng

Mỗi item gồm:

Icon (line icon)
Text label
Padding đều nhau
Danh sách:
Tổng quan (active)
Nhập dữ liệu
Phân tích mạng
Nút quan trọng
Cụm cộng đồng
Điểm nguy cơ
Đề xuất can thiệp
Báo cáo
Trạng thái:
Active:
Nền xanh sáng hơn
Có thể có bo góc nhẹ
Hover:
Đổi màu nền nhẹ (UX chuẩn)
📌 2.4. Khối thông tin dữ liệu (footer sidebar)
Nằm sát đáy sidebar
Dạng text list:
Node
Edge
Loại tương tác
Ngày dữ liệu
Có indicator trạng thái màu xanh:
“Dữ liệu đã sẵn sàng”
Vai trò: cung cấp metadata hệ thống
🧱 3. TOPBAR (THANH TRÊN)
📌 Đặc điểm:
Nền trắng/xám rất nhạt
Chiều cao nhỏ (~50–60px)
Không có logo (đã có ở sidebar)
📌 Thành phần bên phải:
Hướng dẫn
Báo cáo
Xuất dữ liệu
Icon cài đặt ⚙️

👉 Tất cả là dạng text button + icon nhẹ, không gây rối.

📊 4. HÀNG THỐNG KÊ NHANH (METRICS CARDS)
📌 Bố cục:
4 card nằm ngang
Chiếm toàn bộ chiều rộng content
📌 Mỗi card gồm 3 lớp:
(1) Icon
Nằm bên trái
Màu riêng:
Xanh dương
Xanh lá
Cam
Đỏ
(2) Nội dung chính
Số liệu lớn (font rất đậm, nổi bật)
Là focal point của card
(3) Thông tin phụ
% thay đổi
Màu xanh lá → tăng trưởng
📌 Thiết kế:
Nền trắng
Bo góc (~8–12px)
Shadow nhẹ
Khoảng cách giữa các card đều nhau

👉 Phong cách: Material UI / Admin Dashboard

🌐 5. BẢN ĐỒ MẠNG (NETWORK GRAPH PANEL)
📌 Vị trí:
Chiếm khoảng 60–65% chiều ngang
Nằm bên trái khu vực chính
📌 5.1. Header panel
Tiêu đề: “Bản đồ mạng tương tác”
Bên phải:
Nút “Làm mới”
Dropdown “Hiển thị nhãn”
📌 5.2. Legend (chú thích)

Hiển thị dạng chấm màu + label:

Màu	Ý nghĩa
🔴	Nguy cơ cao
🟠	Trung bình
🔵	Thấp
⚪	Chưa xác định

👉 Giúp map trực tiếp với đồ thị

📌 5.3. Đồ thị mạng
Cấu trúc:
Node: hình tròn
Edge: đường nối mảnh
Đặc điểm:
Kích thước node:
Không đồng đều
Node lớn = quan trọng
Màu node:
Theo cụm hoặc mức nguy cơ
Phân cụm rõ:
Các cluster tách biệt bằng khoảng trắng
Layout:
Force-directed (tự cân bằng lực)
Không theo lưới → tự nhiên
📌 5.4. Điều khiển đồ thị
Thanh zoom:
Nút + / -
Có thể có pan (kéo)

🎨 11. NGUYÊN TẮC THIẾT KẾ GIAO DIỆN
📌 11.1. Màu sắc
Primary: xanh đậm (tin cậy, hệ thống)
Semantic colors:
Đỏ → nguy hiểm
Cam → cảnh báo
Xanh → an toàn
📌 11.2. Typography
Font sans-serif hiện đại
Phân cấp:
Số liệu → rất lớn
Tiêu đề → trung bình
Text → nhỏ
📌 11.3. Khoảng cách (Spacing)
Rất đều và nhất quán
Dùng whitespace để phân tách
📌 11.4. Thành phần UI
Card-based design
Badge
Icon + text
Table + graph kết hợp
🧾 12. KẾT LUẬN

Giao diện này là một dashboard phân tích mạng xã hội chuyên sâu, có các đặc trưng:

Phân tầng rõ ràng: Navigation → Overview → Analysis → Detail → Action
Kết hợp:
Trực quan hóa dữ liệu (graph)
Bảng phân tích
Thông tin chi tiết
Hỗ trợ ra quyết định
Thiết kế:
Hiện đại
Trực quan
Tối ưu cho phân tích nhanh
